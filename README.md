# Probability Simulations

This project simulates important probability distributions in Python and compares the simulated results with their theoretical probability functions.

## Distributions

- Binomial distribution
- Poisson distribution
- Normal distribution
- Exponential distribution

## Why This Project Matters

Probability distributions are fundamental in mathematics, statistics, and quantitative finance. They are used to model random events, waiting times, market returns, risk, and uncertainty.

## Technologies Used

- Python
- NumPy
- SciPy
- Matplotlib
